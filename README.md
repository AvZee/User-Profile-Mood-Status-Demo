# User Profile Demo with Mood Status

A full-stack demo of a user profile with an interactive mood status.

## Tech Stack

- React (Vite)
- Hono (API)
- Drizzle ORM
- PostgreSQL
- Zod validation

## Features

- Update user mood and emoji
- Typed API with Hono RPC
- Persistent data with PostgreSQL
- Clean UI with emoji-based profile display
